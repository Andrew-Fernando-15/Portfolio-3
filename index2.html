Samurai OS

Samurai OS is a browser-based desktop environment that simulates an operating system experience. It features a desktop layout with icons, draggable application windows, a taskbar, and built-in tools like a terminal and interactive apps. The project focuses on recreating the feel of a lightweight operating system while running entirely in the browser.

The interface is inspired by classic desktop environments while maintaining a clean and modern Samurai-themed design. Users can open applications inside the desktop, interact with windows, and explore different features without leaving the page.

Tech: HTML, CSS, JavaScript
